🧩 Set Theory

TMS1223 Discrete Mathematics — Foundations of Counting, Relations & Cardinality
📘 Introduction to Sets

A set is an unordered collection of distinct objects. The objects are called elements (or members). If a belongs to set A, write a ∈ A; otherwise a ∉ A. Sets are fundamental for counting, databases, and programming languages.

📌 "Naive set theory" is used here — we focus on practical understanding without formal axioms.
✍️ Describing Sets
Roster (Listing) Method
S = {a, b, c, d}
Order irrelevant, repetitions ignored.
{1,2,3,...} for patterns.
Example: vowels V = {a, e, i, o, u}
Set-Builder Notation
S = { x | P(x) }
Example: O = { x ∈ Z⁺ | x is odd and x<10 }
Interval: [a,b] = {x | a ≤ x ≤ b}
✨ Key sets: ℕ = {0,1,2,…}, ℤ = integers, ℤ⁺ = positive integers, ℝ = real numbers, ℚ = rationals, ℂ = complex numbers.
🌌 Universal Set & Empty Set

Universal set (U): contains everything under consideration (context dependent). Empty set (∅): set with no elements. Note: ∅ ≠ {∅} (the latter contains the empty set as an element).

⚠️ Russell's Paradox: Let S = {x | x ∉ x}. Is S ∈ S? Leads to contradiction. Shows naive set theory has limits, but standard ZFC axioms avoid paradoxes.
💈 Barber paradox: A barber shaves all those who do not shave themselves. Does he shave himself? ⇒ logical paradox.
📏 Subsets & Set Equality

A ⊆ B (A is subset of B) iff every element of A is also in B: ∀x (x∈A → x∈B).

A = B iff A ⊆ B and B ⊆ A.

Proper subset (A ⊂ B): A ⊆ B and A ≠ B (∃x∈B with x∉A).

Theorem 1
For any set S: ∅ ⊆ S and S ⊆ S. Every nonempty set has at least two subsets (∅ and itself).
Counterexample for non-subset
To show A ⊄ B, find x∈A with x∉B.
📐 Example: Set of airline flights from London to New Delhi ⊂ set of all flights (including nonstop). The set of flying squirrels ⊂ set of living creatures that can fly.
🔢 Cardinality & Power Sets

Cardinality |A| = number of distinct elements in a finite set. Infinite sets have infinite cardinality.
Examples: |∅| = 0, |{a,b,c}| = 3, |{{∅}}| = 1, |ℤ| infinite.

⭐ Power set P(A) = set of all subsets of A. If |A| = n, then |P(A)| = 2ⁿ.
Example: A = {a,b} → P(A) = {∅, {a}, {b}, {a,b}}.
Power set examples
{c} → {∅, {c}}
{c,d} → {∅,{c},{d},{c,d}}
{0,{0}} → {∅,{0},{{0}},{0,{0}}}
Ordered n-tuple
(a₁,a₂,…,aₙ) ordered; equality iff each component matches.
Ordered pair (a,b) = (c,d) ⇔ a=c and b=d.
✖️ Cartesian Product & Relations

Cartesian product A × B = {(a,b) | a∈A ∧ b∈B}. Generalization: A₁×A₂×…×Aₙ = {(a₁,…,aₙ) | aᵢ∈Aᵢ}.

Example
A={a,b}, B={1,2,3} →
A×B = {(a,1),(a,2),(a,3),(b,1),(b,2),(b,3)}
Relation
A subset R ⊆ A×B is a relation from A to B. Essential in databases & graph theory.

A×B ≠ B×A unless A=B or one is empty. Cartesian product of 3 sets: A×B×C = triples.

🌐 Example: A={0,1}, B={1,2}, C={0,1,2} → A×B×C has 12 ordered triples (0,1,0), …, (1,2,2).
⚙️ Set Operations
Union (A ∪ B)
{x | x∈A ∨ x∈B}
Example: {1,2,3} ∪ {3,4,5} = {1,2,3,4,5}
Intersection (A ∩ B)
{x | x∈A ∧ x∈B}
Disjoint if A∩B = ∅.
Example: {1,2,3} ∩ {3,4,5} = {3}
Complement (Ā or Aᶜ)
U − A = {x∈U | x∉A}. Example: U=positive integers<100, complement of {x|x>70} is {x|x≤70}.
Difference (A − B)
{x | x∈A ∧ x∉B} = A ∩ B̅.
Example: {1,2,3} − {3,4} = {1,2}.
🎨 Venn diagrams illustrate union, intersection, complement, disjoint sets.
🔐 Set Identities & Proof Techniques

Methods to prove set identities: (1) mutual subset inclusion, (2) set-builder + propositional logic, (3) membership tables.

📜 Second De Morgan Law: Ā ∩ B̅ = A ∪ B   and   A ∩ B = Ā ∪ B̅.
Proof of one De Morgan law (subset method):
Show Ā ∩ B̅ ⊆ A ∪ B: let x∈Ā∩B̅ ⇒ x∉A and x∉B ⇒ x∉(A∪B) ⇒ x∈Ā∪B̅. Reverse inclusion similarly.

🎯 Truth Sets of Predicates

For a predicate P(x) with domain D, the truth set is {x∈D | P(x)}. Quantifiers relate to set operations: ∀xP(x) true iff truth set = D; ∃xP(x) true iff truth set ≠ ∅.

Example
Domain ℤ, P(x): |x|=1 → truth set {-1,1}.
Q(x): x²=2 → truth set ∅.
R(x): |x|=x → truth set ℕ (nonnegatives).
Use in proofs
Set inclusion ↔ universal quantifier: A⊆B ⇔ ∀x(x∈A → x∈B).
🔁 Generalized Unions & Intersections

For indexed collection {A₁, A₂, …, Aₙ}: ⋃_{i=1}^{n} Aᵢ = A₁∪A₂∪…∪Aₙ ; ⋂_{i=1}^{n} Aᵢ = A₁∩A₂∩…∩Aₙ. Union and intersection are associative/commutative.

Example: Aᵢ = {i, i+1, i+2, …} then ⋃Aᵢ = ℕ, ⋂Aᵢ = ∅.

📊 Cardinality & Countable Sets

|A| = |B| if there exists a bijection (one‑to‑one correspondence) between A and B.
|A| ≤ |B| if there exists an injection from A to B.
A set is countable if it is finite or has the same cardinality as ℤ⁺ (positive integers). Countably infinite sets have cardinality ℵ₀ (aleph‑null). Uncountable: ℝ (real numbers).

Examples of countable sets:
• ℤ (integers): list 0,1,-1,2,-2,3,-3,… → bijection
• Odd positive integers: f(n)=2n−1 is a bijection from ℤ⁺ to odd positives.
• ℚ (rationals) is countable (diagonal argument).
📌 One-to-one function (injective): f(a)=f(b) ⇒ a=b. Example: f(x)=x+1 injective. f(x)=x² (over ℤ) not injective.
Onto (surjective): every element in codomain has a preimage.
Bijection = both injective and surjective → used for cardinality comparisons.

Computability: A function is computable if some program computes its values; non‑computable functions exist (e.g., halting problem).

📝 Proving Countability: Odd Positive Integers

Show that the set O = {1,3,5,7,…} is countable. Define f: ℤ⁺ → O by f(n)=2n−1.
• Injective: f(n)=f(m) ⇒ 2n−1=2m−1 ⇒ n=m.
• Surjective: for any odd t, t=2k−1 ⇒ t=f(k). Hence f is bijection ⇒ |O| = |ℤ⁺| ⇒ countable.

✨ This matches the definition: an infinite set is countable iff its elements can be listed as a sequence (a₁,a₂,a₃,…).

🏛️ Historical Notes

∅ introduced by empty set symbol; Venn diagrams by John Venn (1834‑1923); Russell’s paradox (Bertrand Russell, 1872‑1970) showed limitations of naive set theory, leading to axiomatic set theories (ZFC). Descartes (1596‑1650) gave Cartesian coordinates; Cartesian product is named after him. The concept of countability was developed by Georg Cantor (late 19th century), who proved ℝ uncountable.
