🐘 PHP Functions – Part 1

Lecture 5 · ICT1232

🎯 Learning Objectives

After completing this lecture, you should be able to:

  • Identify the use of functions in web development
  • Use user-defined functions in web application development

💡 Introduction to Functions

Definition: A function is a named block of code that performs a specific task. It can accept input values (called parameters), process them, and optionally return a single value.

There are two types of functions in PHP:

🔧 Built-in Functions

Provided by PHP out-of-the-box. No need to define them — just call them. E.g., strlen(), date(), sort()

🛠️ User-Defined Functions

Written by you, the developer, to handle specific tasks in your application.

✅ Advantages of Using Functions

⏱️ Saves Time & Space Write the code once and call it whenever needed — no repetition.
🐛 Improved Reliability Fix a bug in one place (inside the function) instead of hunting for it everywhere it's used.
📖 Better Readability Code is organized into named blocks, making it easier to understand at a glance.

🛠️ User-Defined Functions

Functions defined by the programmer to meet their own requirements. They begin with the keyword function.

Syntax

function functionName() {
    // code to be executed
}

Naming Guidelines

  • Give the function a name that reflects what it does (e.g., calculateTotal(), sendEmail())
  • Must start with a letter or underscore — not a number
  • Can contain letters, numbers, and underscores after the first character
  • Function names are case-sensitive — myFunc() ≠ MyFunc()

📝 Basic Example

<?php
function greetUser() {
    echo "Hello, welcome to PHP functions!";
}

greetUser();   // Calling the function
greetUser();   // Call it as many times as needed
?>
Output:Hello, welcome to PHP functions!
Hello, welcome to PHP functions!

📨 Function Arguments (Parameters)

Arguments are values you send into a function. Inside the function, they are called parameters. Multiple arguments are separated by commas and evaluated left to right.

Syntax

function MyFunction($arg1, $arg2) {
    // use $arg1 and $arg2 here
}

📝 Example — Function with Arguments

<?php
function greet($name, $course) {
    echo "Hello, $name! Welcome to $course.";
}

greet("Alice", "ICT1232");   // Passing 2 arguments
greet("Bob", "Web Dev");
?>
Output:Hello, Alice! Welcome to ICT1232.
Hello, Bob! Welcome to Web Dev.

🔄 Passing Arguments: Value vs Reference

🔵 Passed by Value (Default)
<?php
function addFive($num) {
    $num += 5;
    // Only local copy changes
}

$orignum = 10;
addFive($orignum);
echo "Original: $orignum";
?>
🟣 Passed by Reference (&)
<?php
function addSix(&$num) {
    $num += 6;
    // Original variable changes
}

$orignum = 10;
addSix($orignum);
echo "Original: $orignum";
?>
Output (By Value):Original: 10  ← unchanged
Output (By Reference):Original: 16  ← changed!
Key Difference:
By Value → A copy of the variable is used. The original is not affected.
By Reference → The function gets direct access to the original variable. Changes do affect the original. Use & before the parameter name (e.g., &$num).

⚙️ Default Argument Values

You can assign a default value to a parameter. If the caller doesn't provide a value for that argument, the default is used automatically.

📝 Example

<?php
function setColor($color = "blue") {
    echo "The selected color is: $color";
    echo "<br>";
}

setColor("red");   // Custom value provided
setColor();        // No value → default "blue" is used
setColor("green"); // Another custom value
?>
Output:The selected color is: red
The selected color is: blue
The selected color is: green
💡 Tip: Default parameters must always come after non-default parameters in the function definition. E.g., function myFunc($required, $optional = "default")

📐 Variable Length Argument Functions

Sometimes a function needs to accept a flexible number of arguments. PHP lets you define a function with no parameters at all, and still access whatever arguments are passed using three special functions.

func_num_args()

Returns the total number of arguments passed to the current function.

func_get_arg(index)

Returns a specific argument by its position (0-based index). E.g., index 0 = first argument.

func_get_args()

Returns all arguments as an indexed array. Useful for looping through them.

Rules for variable-length argument functions:
  • The function call can pass more arguments than defined in the function signature
  • You can pass arguments to a function that has no parameters defined at all

📝 Example 1 — func_num_args()

<?php
function printArg() {
    $numargs = func_num_args();          // Count how many args passed
    echo "Number of arguments: $numargs\n";
}

printArg(1, 2, 3);         // Pass 3 values
printArg("a", "b");       // Pass 2 values
?>
Output:Number of arguments: 3
Number of arguments: 2

📝 Example 2 — func_get_arg(position)

<?php
function showArg() {
    $numargs = func_num_args();
    echo "Number of arguments: $numargs\n";

    if ($numargs >= 2) {
        // func_get_arg(1) → gets the 2nd argument (index starts at 0)
        echo "Second argument is: " . func_get_arg(1) . "\n";
    }
}

showArg(1, 5, 3);
?>
Output:Number of arguments: 3
Second argument is: 5

📝 Example 3 — func_get_args()

<?php
function showArray() {
    $numargs = func_num_args();
    echo "Number of arguments: $numargs\n";

    if ($numargs >= 2) {
        echo "Second argument is: " . func_get_arg(1) . "\n";
    }

    $arg_list = func_get_args();   // Get all args as an array
    for ($i = 0; $i < $numargs; $i++) {
        echo "Argument $i is: " . $arg_list[$i] . "\n";
    }
}

showArray(1, 5, 3);
?>
Output:Number of arguments: 3
Second argument is: 5
Argument 0 is: 1
Argument 1 is: 5
Argument 2 is: 3

📝 Example 4 — Full Dynamic Args (for loop + foreach)

<?php
function dynamic_args() {
    // Method 1: for loop with func_get_arg()
    for ($i = 0; $i < func_num_args(); $i++) {
        echo "Argument $i = " . func_get_arg($i) . "<br />";
    }

    // Method 2: foreach with func_get_args()
    $args = func_get_args();
    foreach ($args as $key => $value) {
        echo "Argument {$key}: {$value} <br />";
    }
}

dynamic_args("a", "b", "c", "d", "e");
?>
Output (for loop): Argument 0 = a
Argument 1 = b
Argument 2 = c
Argument 3 = d
Argument 4 = e
Output (foreach): Argument 0: a
Argument 1: b
Argument 2: c
Argument 3: d
Argument 4: e
for loop vs foreach: Both iterate through all arguments. The for loop uses func_get_arg($i) for direct index access, while foreach iterates over the full array returned by func_get_args().

🧪 Putting It All Together

A practical example combining default values, pass by reference, and variable-length args:

<?php
// 1. Default argument value
function introduce($name, $role = "Student") {
    echo "$name is a $role.<br>";
}
introduce("Alice");              // role → default "Student"
introduce("Dr. Bob", "Lecturer"); // role → "Lecturer"

echo "<hr>";

// 2. Pass by reference — double a value
function doubleIt(&$val) {
    $val *= 2;
}
$score = 50;
doubleIt($score);
echo "Score after doubling: $score<br>";  // → 100

echo "<hr>";

// 3. Variable length args — sum any number of values
function sumAll() {
    $total = 0;
    foreach (func_get_args() as $n) {
        $total += $n;
    }
    echo "Total: $total<br>";
}
sumAll(1, 2, 3);           // → Total: 6
sumAll(10, 20, 30, 40);    // → Total: 100
?>
Output: Alice is a Student.
Dr. Bob is a Lecturer.
