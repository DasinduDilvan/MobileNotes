🔌 Opening & Closing a MySQL Connection

To interact with a MySQL database from PHP, you must first open a connection using the mysqli_connect() function.

Return Value Returns a positive integer on success, or FALSE if the connection fails.
Store the Connection Assign the return value to a variable (e.g. $conn) to use it throughout the script.
Close the Connection Always close the connection when done using mysqli_close().

Syntax

$connection = mysqli_connect("host" [, "user", "password" [, "database"]]);
  • host – the hostname where the MySQL server is installed (usually "localhost")
  • user – the MySQL account username
  • password – the MySQL account password
  • database – (optional) the database to select after connecting

Basic Connection Example

$DBConnect = mysqli_connect("localhost", "root", "");

// Close the connection when done
mysqli_close($DBConnect);

🗄️ Creating a Database

After connecting, you can run SQL statements using mysqli_query(). The example below creates a new database named library using the object-oriented style.

<?php
$servername = "localhost";
$username   = "root";
$password   = "";
$dbname     = "library";

// Create connection (OOP style)
$conn = new mysqli($servername, $username, $password);

// Check connection
if ($conn->connect_error) {
    die("Connection failed: " . $conn->connect_error);
}

// Create database
$sql = "CREATE DATABASE $dbname";
if ($conn->query($sql) === TRUE) {
    echo "Database created successfully";
} else {
    echo "Error creating database: " . $conn->error;
}

$conn->close();
?>

Selecting a Database After Connecting (Procedural Style)

<?php
$link = mysqli_connect('localhost', 'mysql_user', 'mysql_password');

if (!$link) {
    die('Not connected: ' . mysqli_error($link));
}

// Select the database
$db_selected = mysqli_select_db($link, 'library');

if (!$db_selected) {
    die("Can't use library: " . mysqli_error($link));
}
?>

📝 CRUD Operations with MySQLi

CREATE TABLE

Use a CREATE TABLE SQL statement with mysqli_query() to create a new table.

<?php
$con = mysqli_connect('localhost', 'root', '', 'library');

if (!$con) {
    die('Connection failed: ' . mysqli_connect_error());
}

$sql = "CREATE TABLE IF NOT EXISTS book (
    id           INT AUTO_INCREMENT PRIMARY KEY,
    Title        VARCHAR(100),
    Author       VARCHAR(100),
    No_of_copies INT
)";

if (mysqli_query($con, $sql)) {
    echo "Table 'book' created successfully";
} else {
    echo "Error creating table: " . mysqli_error($con);
}

mysqli_close($con);
?>
PRIMARY KEY Indicates a field (or fields) used as a referential index for the table. Each value must be unique.
AUTO_INCREMENT Automatically assigns the next sequential integer value to the field. No manual entry needed.
NOT NULL Forces the field to always contain data — it cannot be left empty.

INSERT – Adding Records

<?php
$con = mysqli_connect('localhost', 'root', '', 'library');

if (!$con) {
    die('Could not connect: ' . mysqli_connect_error());
}

$sql = "INSERT INTO book (Title, Author, No_of_copies)
        VALUES ('PHP Basics', 'Longbottom', 35)";

if (mysqli_query($con, $sql)) {
    echo "Record inserted successfully";
} else {
    echo "Error: " . mysqli_error($con);
}

mysqli_close($con);
?>

UPDATE – Updating Records

<?php
$con = mysqli_connect('localhost', 'root', '', 'library');

if (!$con) {
    die('Connection failed: ' . mysqli_connect_error());
}

$sql = "UPDATE book
        SET Title = 'Advanced PHP'
        WHERE Title = 'PHP Basics'";

if (mysqli_query($con, $sql)) {
    echo "Book title updated successfully";
} else {
    echo "Error updating record: " . mysqli_error($con);
}

mysqli_close($con);
?>

📥 Retrieving Records

mysqli_fetch_row() — Indexed Array Returns fields in the current row as an indexed array (accessed by numeric index: $row[0], $row[1], etc.) and moves the result pointer to the next row.
mysqli_fetch_assoc() — Associative Array Returns fields in the current row as an associative array (accessed by field name: $Row['Title']) and moves the result pointer to the next row.

Example: mysqli_fetch_row()

while ($row = mysqli_fetch_row($result1)) {
    echo "<tr>";
    echo "<td>" . $row[0] . "</td>";  // id
    echo "<td>" . $row[1] . "</td>";  // Title
    echo "<td>" . $row[2] . "</td>";  // Author
    echo "<td>" . $row[3] . "</td>";  // No_of_copies
    echo "</tr>";
}

Example: mysqli_fetch_assoc()

<?php
$con = mysqli_connect("localhost", "root", "", "library");

if (!$con) {
    die('Could not connect: ' . mysqli_error($con));
}

$q      = "SELECT * FROM book";
$result = mysqli_query($con, $q);

while ($Row = mysqli_fetch_assoc($result)) {
    echo $Row['Title'];
    echo $Row['Author'];
    echo $Row['No_of_copies'];
}

mysqli_close($con);
?>
A result pointer is a special type of variable that refers to the currently selected row in a result set. Each call to mysqli_fetch_row() or mysqli_fetch_assoc() automatically advances the pointer to the next row.

📊 Accessing Query Result Information

mysqli_num_rows($result) Returns the number of rows in a query result. Accepts the result variable as argument.
mysqli_num_fields($result) Returns the number of fields (columns) in a query result. Accepts the result variable as argument.

Example

$SQLstring = "SELECT * FROM cars";
$result    = mysqli_query($con, $SQLstring);

$NumRows   = mysqli_num_rows($result);
$NumFields = mysqli_num_fields($result);

if ($NumRows != 0 && $NumFields != 0)
    echo "Your query returned " . $NumRows . " rows and " . $NumFields . " fields.";
else
    echo "Your query returned no results.";

mysqli_close($con);

🔢 mysqli_affected_rows() & mysqli_info()

mysqli_affected_rows($conn) Returns the number of rows affected by the last INSERT, UPDATE, or DELETE query. Useful to confirm how many records were changed.
mysqli_info($conn) Returns a string with operation counts (records inserted/updated, duplicates, warnings) depending on the type of query executed.

Example: mysqli_affected_rows()

$QueryResult = mysqli_query($con,
    "UPDATE book SET No_of_copies='20'
     WHERE Title='PHP Basics' AND Author='Longbottom'");

if ($QueryResult == FALSE)
    echo "Unable to execute the query. Error: " . mysqli_error($con);
else
    echo "Successfully updated " . mysqli_affected_rows($con) . " record(s).";

mysqli_close($con);

Example: mysqli_info()

$sql2 = "INSERT INTO book VALUES
    ('0077862589', 'Organizational behavior', 'Shane Steven', 10),
    ('0321573513', 'Algorithms', 'Kevin Wayne', 15)";

$result2 = mysqli_query($con, $sql2);

if (!$result2) {
    echo "Error code " . mysqli_errno($con) . ": " . mysqli_error($con);
} else {
    echo "Successfully added the record.";
    echo mysqli_info($con); // e.g. "Records: 2 Duplicates: 0 Warnings: 0"
}

mysqli_close($con);
Sample Output of mysqli_info():
Records: 2 Duplicates: 0 Warnings: 0

📚 MySQLi Function Reference

Function Description
mysqli_connect() Opens a connection to a MySQL database server. Returns a connection object or FALSE on failure.
mysqli_close() Closes an open database connection.
mysqli_select_db() Selects a database to use on the connected server.
mysqli_query() Sends an SQL statement to MySQL for execution. Returns a result object or TRUE/FALSE.
mysqli_errno() Returns the error code from the last attempted MySQL function call, or zero if no error occurred.
mysqli_error() Returns the error message from the last attempted MySQL function call, or an empty string if no error.
mysqli_connect_error() Returns the error description from the last connection attempt.
mysqli_fetch_row() Returns the current row of a result set as an indexed array, then advances the pointer.
mysqli_fetch_assoc() Returns the current row of a result set as an associative array (keyed by field name), then advances the pointer.
mysqli_free_result() Closes/frees a result set to release memory.
mysqli_num_rows() Returns the number of rows in a query result.
mysqli_num_fields() Returns the number of fields (columns) in a query result.
mysqli_affected_rows() Returns the number of rows affected by the last INSERT, UPDATE, or DELETE statement.
mysqli_info() Returns a string with the count of records affected, duplicates, and warnings for the last query.

📌 Summary

Connect & Close mysqli_connect() opens the connection. mysqli_close() closes it. Always close after use.
Error Handling mysqli_errno() returns the error code. mysqli_error() returns the error message. Use with die() to halt on failure.
Execute SQL mysqli_query() sends any SQL statement. Works for SELECT, INSERT, UPDATE, DELETE, CREATE.
Fetch Results mysqli_fetch_row() → indexed array. mysqli_fetch_assoc() → associative array by field name.
Result Info mysqli_num_rows() counts rows. mysqli_num_fields() counts columns. mysqli_affected_rows() counts changes.
Table Clauses PRIMARY KEY – referential index. AUTO_INCREMENT – auto sequential value. NOT NULL – field must have data.
