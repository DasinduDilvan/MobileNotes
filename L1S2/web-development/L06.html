1. Server-Side Scripting

Server-side scripting is a web server technology in which a user's request is fulfilled by running a script directly on the web server to generate dynamic web pages.

Used to provide interactive web sites that interface to databases or other data stores on the server.

2. What is PHP?

Origin Conceived in 1994 by Rasmus Lerdorf.
Full Form PHP: Hypertext Preprocessor (recursive acronym).
Nature PHP pages contain HTML with embedded code that performs server-side logic.
Delimiters PHP code is enclosed in <?php … ?> tags to switch in and out of PHP mode.

Basic PHP in HTML Example:

<!DOCTYPE html>
<html>
<body>
<?php
    echo "My first PHP script!";
?>
</body>
</html>

4. PHP Variables

A variable is a representation of a particular value. PHP variables can hold values or expressions.

PHP has no command for declaring a variable — it is created the moment you first assign a value to it.

Examples:

$txt = "Hello world!";
$x   = 5;

Rules for PHP Variables

  • A variable starts with the $ sign, followed by the name of the variable.
  • A variable name must start with a letter or an underscore character.
  • A variable name cannot start with a number.
  • A variable name can only contain alpha-numeric characters and underscores (A–z, 0–9, and _ ).
  • Variable names are case-sensitive — $age and $AGE are two different variables.

Example — Variables in expressions:

<?php
$num1 = 5;
$num2 = 6;
echo "Number 1 is ".$num1."<br/>";
echo "Number 2 is ".$num2."<br/>";
// addition
$result = $num1 + $num2;
echo "Result is ".$result; // holds value of expression
?>

5. PHP Variable Scope

The scope of a variable is the part of the script where the variable can be referenced or used. PHP has three different variable scopes:

local Declared inside a function. Only accessible within that function.
global Declared outside a function. Only accessible outside functions by default.
static A local variable that is not deleted after the function finishes executing.

Global Scope Example

<?php
$x = 5; // global scope

function myTest() {
  // $x is NOT accessible here
  echo $x; // generates error
}
myTest();
echo $x; // accessible here → 5
?>

Local Scope Example

<?php
function myTest() {
  $x = 5; // local scope
  echo $x; // accessible here → 5
}
myTest();
// $x NOT accessible here
echo $x; // generates error
?>

The global Keyword — used to access a global variable from within a function:

<?php
$x = 5;
$y = 10;

function myTest() {
    global $x, $y;
    $y = $x + $y;
}
myTest();
echo $y; // outputs 15
?>

The static Keyword — prevents a local variable from being deleted after function execution:

<?php
function myTest() {
    static $x = 0;
    echo $x;
    $x++;
}
myTest(); // outputs 0
myTest(); // outputs 1
myTest(); // outputs 2
?>

✏️ Exercise — What is the output?

function updateCounter() {
    static $counter = 0;
    $counter++;
    echo "Static counter is now {$counter}<br>";
}

$counter = 10;
updateCounter();  // Static counter is now 1
updateCounter();  // Static counter is now 2
echo "Global counter is {$counter}\n"; // Global counter is 10
The static $counter inside the function is separate from the global $counter = 10. Static variables retain their value across calls but remain local to the function.

9. Conditional Statements

Used to perform different actions for different conditions. PHP supports the following conditional statements:

if — Execute code only if a condition is true
if…else — One code block if true, another if false
if…elseif…else — Select one from several blocks
switch — Select one of many blocks to execute based on a value

9.1 if Statement

<?php
$num = 0;
if ($num == 0) {
    echo "Number is equal to 0";
}
?>

9.2 if…else Statement

<?php
$num = 0;
if ($num == 0) {
    echo "Number is equal to 0";
} else {
    echo "Number is not equal to 0";
}
?>

9.3 if…elseif…else Statement

<?php
$num = 0;
if ($num > 5) {
    echo "Number is greater than 5";
} elseif ($num > 2) {
    echo "Number is greater than 2";
} else {
    echo "Number is 0 or less than 0";
}
?>

9.4 switch Statement

switch($name) {
    case 'John':
        echo "Name is John";
        break;
    case 'Mary':
        echo "Name is Mary";
        break;
    default:
        echo "No name to display";
        break;
}

📋 Summary

Features of PHP Free, open source, platform independent, supports MySQL, compatible with major servers.
Basic Syntax PHP embedded in HTML using <?php … ?> tags. Statements end with ;.
Running PHP Requires a web server (Apache/IIS) with PHP installed. Accessed via a browser.
PHP Variables Start with $. No declaration needed. Created on first assignment. Case-sensitive.
Variable Scope Three scopes: local, global, static. Use global keyword to access globals inside functions.
Data Types Scalar (bool, int, float, string), Compound (array, object), Special (resource, NULL).
Operators Arithmetic, Assignment, Comparison, Increment/Decrement, Logical, Concatenation (.).
Conditional Statements if, if…else, if…elseif…else, switch for decision-making logic.
