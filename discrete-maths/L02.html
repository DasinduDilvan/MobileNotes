1. Conditional Statements (Implication)  →

Definition: Let p and q be propositions. The conditional statement p → q is the proposition "if p, then q."

p → q is false only when p is true and q is false. It is true in all other cases.

In p → q:   p = hypothesis (antecedent / premise)    q = conclusion (consequence).
A conditional statement is also called an implication.

Type: Binary Operator  |  Symbol: →

pqp → q

All ways to express p → q in English
  • • "if p, then q"
  • • "p implies q"
  • • "if p, q"
  • • "p only if q"
  • • "p is sufficient for q"
  • • "a sufficient condition for q is p"
  • • "q if p" / "q whenever p" / "q when p"
  • • "q is necessary for p"
  • • "a necessary condition for p is q"
  • • "q follows from p"
  • • "q unless ¬p"

Examples

Example 1 p: "Maria learns discrete mathematics."
q: "Maria will find a good job."

p → q: "If Maria learns discrete mathematics, then she will find a good job."
Also: "Maria will find a good job when she learns discrete mathematics."
Example 2 p: "x = 2"    q: "x² = 4"

p → q: "If x = 2, then x² = 4."
Also: "x² = 4 when x = 2."
"x = 2 unless x² ≠ 4."

2. Converse, Contrapositive & Inverse of p → q

Converse q → p
Swap the hypothesis and conclusion.
Contrapositive ¬q → ¬p
Negate both and swap them.
Inverse ¬p → ¬q
Negate both (without swapping).

Equivalent Propositions: Two compound propositions that always have the same truth value are called logically equivalent.

  • ✅ p → q ≡ ¬q → ¬p   (Conditional ≡ Contrapositive)
  • ✅ q → p ≡ ¬p → ¬q   (Converse ≡ Inverse)
  • ❌ p → q is NOT equivalent to its converse or inverse.
Example — "The home team wins whenever it is raining."

Rewritten as: p → q where p = "It is raining", q = "The home team wins".

  • Original (p → q): "If it is raining, then the home team wins."
  • Contrapositive (¬q → ¬p): "If the home team does not win, then it is not raining."
  • Converse (q → p): "If the home team wins, then it is raining."
  • Inverse (¬p → ¬q): "If it is not raining, then the home team does not win."

3. Biconditional (if and only if)  ↔

Definition: Let p and q be propositions. The biconditional statement p ↔ q is the proposition "p if and only if q."

p ↔ q is true when p and q have the same truth values, and false otherwise. Also called bi-implications.

Type: Binary Operator  |  Symbol: ↔

pqp ↔ q
TTT
TFF
FTF
FFT
Ways to express p ↔ q
  • • "p is necessary and sufficient for q"
  • • "if p then q, and conversely"
  • • "p iff q"

Example

p: "You can take the flight."   q: "You buy a ticket."
p ↔ q: "You can take the flight if and only if you buy a ticket."
True when both hold or both fail. False when one holds without the other.

4. Precedence of Logical Operators

When parentheses are absent, operators are applied in the following order (lower number = higher precedence):

OperatorPrecedence
¬1 (highest)
∧2
∨3
→4
↔5 (lowest)
Key Rules
  • • ¬ is applied before all other operators.
  • • ∧ takes precedence over ∨.
  • • → takes precedence over ↔.
Example: p ∨ q ∧ r is read as p ∨ (q ∧ r), not (p ∨ q) ∧ r.

6. Logic and Bit Operations

Bit: A symbol with two possible values — 0 (zero) and 1 (one). The word bit comes from binary digit. Introduced by statistician John Tukey in 1946.

Convention:  1 represents True (T)    0 represents False (F)

A variable is called a Boolean variable if its value is either true or false. Computer bit operations correspond directly to logical connectives.

Bit Strings

Definition: A bit string is a sequence of zero or more bits. The length of the string is the number of bits it contains.
Example: 101010011 is a bit string of length 9.

The bitwise OR (∨), bitwise AND (∧), and bitwise XOR (⊕) of two equal-length bit strings are computed by applying the respective operation to each corresponding pair of bits.

  • b) (p ∨ q) ∨ r ≡ p ∨ (q ∨ r)    (Associativity of OR)

Exercise 3 — Translate Using p and q

Let p: "It is below freezing."   q: "It is snowing."

  • a) It is below freezing and snowing. → p ∧ q
  • b) It is below freezing but not snowing. → p ∧ ¬q
  • c) It is not below freezing and it is not snowing. → ¬p ∧ ¬q
  • d) It is either snowing or below freezing (or both). → p ∨ q
  • e) If it is below freezing, it is also snowing. → p → q
  • f) Either below freezing or snowing, but not snowing if below freezing. → p ⊕ q
  • g) Below freezing is necessary and sufficient for snowing. → p ↔ q
