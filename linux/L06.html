Functions
Definition: Function performs a particular activity in shell scripting. 
function-name ()
{
   command ...
   return
}

HelloUsr()
{
echo "Hello $LOGNAME, Have a nice day!"
return
}

$ HelloUsr
Hello student, Have a nice day!
Passing Parameters to Function
function calc()
{
n1=$1
op=$2
n2=$3

if [ $# -eq 3 ]; then
ans=`expr $n1 $op $n2`
echo "$n1 $op $n2 = $ans"
return $ans
else
echo "Function calc requires at least three args"
fi
}

calc 6 + 5
calc 6 - 5
calc 6 / 2
$# → Number of arguments passed
The shift Command
  • Moves positional parameters to the left
  • Used to process command-line arguments
Example: 
$1=one $2=two $3=three
shift

Now:
$1=two $2=three

shift 2
Shell Configuration Files
  • .bashrc → executed when new shell starts
  • .bash_logout → executed when logging out
  • .bash_profile → contains environment variables
Only root can edit /etc/bashrc
/dev/null

command > /dev/null

ls > /dev/null

Dialog Utility
Dialog allows creating terminal-based UI dialog boxes.
  • yes/no box
  • msgbox
  • infobox
  • inputbox
  • textbox
  • menu
sudo apt install dialog
Dialog Examples
dialog --msgbox "Hello, world!" 5 20
dialog --yesno "Are you having fun?" 6 25
dialog --inputbox "Enter your name" 8 40
dialog --textbox /etc/profile 22 70
Menu & Checklist
dialog --menu "Choose one:" 10 30 3 \
1 red 2 green 3 blue

dialog --checklist "Choose toppings:" 10 40 3 \
1 Cheese on \
2 "Tomato Sauce" on
Confirmation Box
Exit Status:
  • 0 → Yes
  • 1 → No
  • 255 → ESC
dialog --yesno "Delete file?" 7 50
Input Box
dialog --inputbox "Enter name" 8 60 2>file
Time & Calendar Dialogs
dialog --timebox "Set time" 0 0
dialog --calendar "Select date" 0 0
