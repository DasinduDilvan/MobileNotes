📦 What Are Built-in Functions?

Key Point: Built-in functions are ready-made functions provided by PHP. You do not need to define them — just call them by name and pass the required values.
Convention: By convention, most PHP built-in functions are called with all lowercase letters (e.g., strlen(), date(), sort()).

PHP built-in functions are grouped by category:

🔤 String FunctionsHandle and manipulate text strings
📋 Array FunctionsCreate and manipulate arrays
📅 Date / Time FunctionsExtract and format date & time from server
🗄️ MySQL FunctionsConnect to and query MySQL databases
🔢 Math FunctionsPerform mathematical operations
✉️ Mail FunctionsSend emails from PHP scripts

🔤 String Functions

String functions allow you to manipulate and process text values.

FunctionDescription
echo()Output one or more strings to the browser
explode(separator, string, limit)Breaks a string into an array by a separator
str_replace(find, replace, string, count)Replaces some characters with other characters in a string
strlen(string)Returns the length (number of characters) of a string
trim(string, charlist)Strips whitespace (or specified characters) from both sides of a string
strrev(string)Reverses a string

📝 Examples

1. echo() — Output text 
<?php
echo("Hello, World!");           // Simple string
echo("Hello" . " " . "PHP!");   // Concatenation with dot
?>
Output:Hello, World!
Hello PHP!
2. explode() — Break string into array 
<?php
$fruits = "apple,banana,cherry";
$arr = explode(",", $fruits);
print_r($arr);   // Splits by comma
?>
Output:Array ( [0] => apple [1] => banana [2] => cherry )
3. str_replace() — Find & replace in string 
<?php
$text = "I love cats and cats love me.";
$new = str_replace("cats", "dogs", $text);
echo $new;
?>
Output:I love dogs and dogs love me.
4. strlen() — Get string length 
<?php
$str = "Hello PHP";
echo strlen($str);   // Counts all characters including space
?>
Output:9
5. trim() — Strip whitespace 
<?php
$str = "   Hello PHP   ";
echo trim($str);   // Removes spaces from both ends
?>
Output:Hello PHP
6. strrev() — Reverse a string 
<?php
echo strrev("madam");   // Reverses the string
echo "<br>";
echo strrev("PHP");
?>
Output:madam
PHP

📋 Array Functions

Array functions allow you to create and manipulate arrays. PHP supports both simple (1D) and multi-dimensional arrays.

FunctionDescription
array()Creates an array
current()Returns the current element (pointed to by the internal pointer) in an array
next()Advances the internal array pointer forward by one
reset()Sets the internal pointer of an array back to its first element
sort()Sorts an array in ascending order

📝 Examples

1. array() — Create an array 
<?php
$colors = array("red", "green", "blue");
echo $colors[0];   // Access first element (index 0)
print_r($colors);  // Print full array structure
?>
Output:red
Array ( [0] => red [1] => green [2] => blue )
2. current(), next(), reset() — Traverse array with pointer 
<?php
$items = array("apple", "banana", "cherry");

echo current($items); // Returns current element → apple
echo "<br>";
next($items);          // Move pointer forward
echo current($items); // Now returns → banana
echo "<br>";
reset($items);         // Reset pointer to start
echo current($items); // Back to → apple
?>
Output:apple
banana
apple
3. sort() — Sort an array 
<?php
$nums = array(3, 1, 5, 2, 4);
sort($nums);        // Sorts in ascending order
print_r($nums);
?>
Output:Array ( [0] => 1 [1] => 2 [2] => 3 [3] => 4 [4] => 5 )

📅 Date / Time Functions

Date/Time functions allow you to extract and format the date and time from the server.

Syntax: date(format, timestamp)
The format is a string of characters that define how the date/time is displayed. The timestamp parameter is optional — if omitted, it uses the current server time.

Format Characters Reference

CharacterDescriptionExample
dDay of the month (01–31)09
DShort day name (3 letters)Mon
mMonth number (01–12)03
MShort month name (3 letters)Mar
yYear — 2 digits25
YYear — 4 digits2025
zDay of the year (0–365)68
HHour in 24-hour format (00–23)14
hHour in 12-hour format (01–12)02
iMinutes with leading zeros (00–59)05
sSeconds with leading zeros (00–59)47
aLowercase am or pmam

📝 Example — Date

<?php
// date(format, timestamp)
echo(date("d") . "<br />");   // Day number → 09
echo(date("D") . "<br />");   // Short day  → Mon
echo(date("m") . "<br />");   // Month num  → 03
echo(date("M") . "<br />");   // Short month→ Mar
echo(date("y") . "<br />");   // Year 2-dig → 25
echo(date("Y") . "<br />");   // Year 4-dig → 2025
?>
Output (example):09
Mon

📝 Example — Time

<?php
// z - The day of the year (0 through 365)
echo(date("z") . "<br />");

// Display full time: HH:MM:SS am/pm
// H - 24-hour, i - minutes, s - seconds, a - am/pm
echo(date("H") . ":" . date("i") . ":" . date("s") . " " . date("a") . "<br>");
?>

🗄️ MySQL Functions

MySQL functions allow PHP to connect to and interact with MySQL database servers.

⚠️ Note: The mysql_* functions shown here are the older procedural style. In modern PHP (7+), these are deprecated. Use mysqli_* or PDO instead. However, understanding these is still useful for reading legacy code.
FunctionDescription
mysql_connect(server, user, pwd)Opens a connection to a MySQL server
mysql_close(connection)Closes a MySQL connection
mysql_error(connection)Returns the error description from the last MySQL operation
mysql_select_db(database, connection)Sets the active MySQL database to use
mysql_query(query, connection)Executes an SQL query on the database
mysql_result(data, row, field)Returns the value of a specific field in a result set
mysql_fetch_assoc(data)Returns a row from the result set as an associative array
mysql_fetch_row(data)Returns a row from the result set as a numeric (indexed) array

📝 Full Example — Connect & Query

<?php
// Step 1: Open connection to MySQL server
$con = mysql_connect("localhost", "mysql_user", "mysql_pwd");

// Step 2: Check if connection failed
if (!$con) {
    die('Could not connect: ' . mysql_error());
}

// Step 3: Select the database
$db_selected = mysql_select_db("test_db", $con);

// Step 4: Write and run an SQL query
$sql = "SELECT * FROM Person";
$result = mysql_query($sql, $con);

// Step 5: Get data from result
echo mysql_result($result, 0);          // Value of first field in first row
print_r(mysql_fetch_assoc($result));   // Row as associative array
print_r(mysql_fetch_row($result));     // Row as indexed array

// Step 6: Close connection
mysql_close($con);
?>

Understanding the Steps

Step 1–2Connect to server, check for errors using die()
Step 3Select which database to work with
Step 4Run an SQL SELECT query
Step 5Retrieve and display data from result
Step 6Always close the connection when done
Associative vs Indexed Array:
mysql_fetch_assoc() → Returns ["name" => "Alice", "age" => 20] (key = column name)
mysql_fetch_row() → Returns [0 => "Alice", 1 => 20] (key = column number)

📂 Programming in Multiple Files

When programs grow large, it's best practice to split code across multiple files. Common code (e.g., headers, footers, database connections) can be written once and reused everywhere.

How it works: Including a file produces the same result as if you copied the code from that file and pasted it exactly at the location where the include/require statement is.
include()

Includes and evaluates a file. If the file is not found, PHP shows a warning but continues execution.

include "header.php";
require()

Includes and evaluates a file. If the file is not found, PHP shows a fatal error and stops execution.

require "header.php";

include_once() and require_once()

These work the same as include() and require(), but they only include the file once — even if the statement is encountered multiple times. This prevents re-declaring functions or classes.

include_once "header.php";
require_once "header.php";

📝 Example — Reusable Header

/* header.php */
<?php
echo "<h1>My Website</h1>";
echo "<nav>Home | About | Contact</nav>";
?>

/* index.php */
<?php
include_once "header.php";  // Reuse header on every page
echo "<p>Welcome to the homepage!</p>";
?>
include / include_onceUse when the file is optional — page may still work without it
require / require_onceUse when the file is essential — stop execution if missing (e.g., config, DB connection)

🧪 Exercises

✏️ Exercise 01 — Factorial of a Number

Write a function to calculate the factorial of a non-negative integer. The function accepts the number as an argument.

Concept: Factorial of n = n × (n−1) × (n−2) × ... × 1. By definition, 0! = 1.

<?php
function factorial_of_a_number($n) {
    if ($n == 0) {
        return 1;       // Base case: 0! = 1
    } else {
        return $n * factorial_of_a_number($n - 1);  // Recursive call
    }
}

print_r(factorial_of_a_number(4) . "\n");  // 4! = 4×3×2×1 = 24
print_r(factorial_of_a_number(5) . "\n");  // 5! = 5×4×3×2×1 = 120
print_r(factorial_of_a_number(0) . "\n");  // 0! = 1
?>
Output:24
120
1
How it works (Recursion Trace for n=4):
factorial(4) → 4 × factorial(3)
factorial(3) → 3 × factorial(2)
factorial(2) → 2 × factorial(1)
factorial(1) → 1 × factorial(0)
factorial(0) → 1  ← base case reached!
Result: 4 × 3 × 2 × 1 × 1 = 24

✏️ Exercise 02 — Palindrome Checker

Write a PHP function that checks whether a passed string is a palindrome or not.

What is a Palindrome? A word, phrase, or sequence that reads the same backward as forward. Example: madam, racecar, level. 
<?php
function check_palindrome($string) {
    if ($string == strrev($string))   // Compare string with its reverse
        return 1;   // 1 means TRUE — it is a palindrome
    else
        return 0;   // 0 means FALSE — not a palindrome
}

echo check_palindrome('madam') . "\n";    // 1 (palindrome)
echo check_palindrome('racecar') . "\n"; // 1 (palindrome)
echo check_palindrome('hello') . "\n";   // 0 (not a palindrome)
?>
Output:1
1
0
Key function used: strrev($string) — reverses the string.
The function compares the original with its reversed version. If they match → palindrome!
