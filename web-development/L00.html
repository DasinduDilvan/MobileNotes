📦 var / let / const
varFunction-scoped. Avoid — causes unexpected bugs. Accessible outside loops.
letBlock-scoped. Use when reassignment needed.
constBlock-scoped + no reassign. Use by default. Objects can still be mutated.
var x = 15;         // function scope (OLD)
let fruit = 'banana'; // block scope
const PI = 3.14;    // cannot reassign

const list = [1, 2, 3];
list.push(4); // OK — mutating allowed
// list = [];  // ERROR — reassign not allowed

// var leaks out of loops:
for (var i = 0; i < 3; i++) {}
console.log(i); // 3 (works — var is function scope)

// let does NOT:
for (let j = 0; j < 3; j++) {}
// console.log(j); // ERROR — j not defined
Best practice: const always, let if reassign needed, never var.

✏️ Create, Write, Read, Close
<?php
// CREATE / OPEN
$filename = "myfile.txt";
$handle = fopen($filename, 'w') or die('Cannot open: ' . $filename);

// WRITE
fwrite($handle, "Hello, file!\n");
fwrite($handle, "Second line\n");

// CLOSE — always do this!
fclose($handle);

// READ (open with 'r', use filesize for full read)
$file = fopen($filename, "r") or die("Error opening");
$size = filesize($filename);           // get byte size
$text = fread($file, $size);           // read whole file
fclose($file);
echo "<pre>" . $text . "</pre>";  // preserve formatting

// APPEND — use 'a' mode, won't erase existing content
$f = fopen($filename, "a");
fwrite($f, "Appended line\n");
fclose($f);
?>
🔁 Read Line-by-Line with fgets() + feof()
<?php

Lecture 11 — PHP Sessions

🔐 What are Sessions?
Sessions — stored server-side. Secure. Temporary (lost when browser closes).
Cookies — stored client-side (browser). Less secure. Can persist.
Critical Rule: session_start() must be called at the very TOP of EVERY page — before any HTML output or whitespace.
⚙️ Session Lifecycle
<?php
// page1.php — START + SET
session_start(); // MUST be first line

$_SESSION["username"] = "JohnDoe";
$_SESSION["views"]    = 1;
$_SESSION["cart"]     = "Laptop";
echo "Session set!";
?>

<?php
// page2.php — READ session variables
session_start();
echo "Welcome " . $_SESSION["username"];
echo "Views: "  . $_SESSION["views"];
?>
🧪 Practice — Hit Counter (Increment views)
<?php
session_start();

if (isset($_SESSION["views"])) {
  $_SESSION["views"]++;       // increment on each load
} else {
  $_SESSION["views"] = 1;    // first visit
}

echo "Page views: " . $_SESSION["views"];
?>
🗑️ Removing Session Data
<?php
session_start();

// Remove ONE variable
unset($_SESSION['views']);

// Remove ALL variables (session still exists)
session_unset();

// DESTROY the entire session
session_destroy();
?>
FunctionEffect
session_start()Start/resume session. MUST be first.
$_SESSION["key"] = valStore data
$_SESSION["key"]Read data
isset($_SESSION["key"])Check if variable exists
unset($_SESSION["key"])Delete one variable
session_unset()Delete all variables
session_destroy()Destroy entire session

✏️ Create, Read, Check, Delete
<?php

<?php
// READ cookie (on next page load)
echo $_COOKIE["name"] . "<br>";  // John Watson

// CHECK if cookie exists with isset()
if (isset($_COOKIE["name"])) {
  echo "Welcome " . $_COOKIE["name"];
} else {
  echo "Not recognized";
}

// READ ALL cookies
echo "Total: " . count($_COOKIE) . "<br>";
foreach ($_COOKIE as $key => $value) {
  echo "Cookie: " . $key . " = " . $value . "<br>";
}

// DELETE cookie — set expiry in the PAST
setcookie("name", "", time() - 60, "/", "", 0);
setcookie("age",  "", time() - 60, "/", "", 0);
?>

<?php
class Player {
  public $name;     // member variable (attribute)
  public $health;

  // __construct — called automatically on new Player()
  public function __construct($name, $startingHealth = 100) {

    $this->health = $startingHealth;

  }

    return "{$this->name} has {$this->health} HP.";
  }
}

?>

<?php
class BankAccount {

    if ($amount > 0) {

    }
  }

  public function getBalance() {
    return "Balance: $" . $this->balance;
  }

$acc = new BankAccount();

🔀 Polymorphism · 🎭 Abstraction · 📌 Static
<?php
// POLYMORPHISM — same method name, different behavior
interface Shape {
  public function calculateArea();
}
class Circle implements Shape {
  public function calculateArea() { return "Calculating πr²"; }
}
class Square implements Shape {
  public function calculateArea() { return "Calculating side²"; }
}
$shapes = [new Circle(), new Square()];
foreach ($shapes as $s) { echo $s->calculateArea() . "\n"; }

// STATIC — access without creating object; use self:: inside class
class Counter {
  public static $count = 0;

  public static function increment() {
    self::$count++;           // self:: not $this
  }
}
Counter::increment();          // call without 'new'
Counter::increment();
echo Counter::$count;          // 2
?>
$this vs self: $this->property for instances. self::$property for static class members.

📝 Step 1 — HTML Form (Blade View)
<!-- resources/views/home.blade.php -->
<form action="{{ route('formsubmitted') }}" method="POST">
  @csrf    <!-- REQUIRED for ALL POST forms! -->

  <label>Name:</label>
  <input type="text" name="user_name" required>

  <label>Email:</label>
  <input type="email" name="user_email" required>

@csrf is mandatory in every POST form. Without it, Laravel returns a 419 error. It prevents Cross-Site Request Forgery attacks.
📥 Step 2 — Handle Form in web.php
// routes/web.php
use Illuminate\Http\Request;

Route::post('/formsubmitted', function (Request $request) {
  // $request->input('field_name') reads form input by name attribute
  $fname = $request->input('user_name');
  $email = $request->input('user_email');

  return "Your name is {$fname} and your email is {$email}";
})->name('formsubmitted');
🛡️ CSRF Protection
What is CSRF?Cross-Site Request Forgery — a hacker's page secretly submits a form to your app using the logged-in user's browser session.
Laravel's solutionGenerates a secret token per session. Verifies it on every POST/PUT/PATCH/DELETE request.

  ...
</form>

⚙️ Step 5 — Controller Methods (PostController.php)
// index() — list all posts
public function index() {
  $posts = Post::all();
  return view('posts.index', compact('posts'));
}

// create() — show create form
public function create() {
  return view('posts.create');
}

// store() — validate & save new post
public function store(Request $request) {
  $validated = $request->validate([
    'title' => 'required',
    'body'  => 'required',
  ]);
  Post::create($validated);
  return redirect()->route('posts.index')
    ->with('success', 'Post created successfully');
}

// edit() — show edit form
public function edit($id) {
  $post = Post::findOrFail($id);  // 404 if not found
  return view('posts.edit', compact('post'));
}

// update() — validate & save edited post
public function update(Request $request, $id) {
  $validated = $request->validate([
    'title' => 'required',
    'body'  => 'required',
  ]);
  $post = Post::findOrFail($id);
  $post->update($validated);
  return redirect()->route('posts.index')
    ->with('success', 'Post updated successfully');
}

// destroy() — delete a post
public function destroy($id) {
  $post = Post::findOrFail($id);
  $post->delete();
  return redirect()->route('posts.index')
    ->with('success', 'Post deleted successfully');
}
