💡 What Is an Array?

Definition: An array is a special variable that can hold more than one value at a time. It is an ordered collection of key-value pairs, where each value is called an element.
  • Stores a group of related data under a single variable name
  • Each element is accessed by its index (key)
  • Can store heterogeneous data — strings, integers, booleans, even other arrays
📌 Indexed ArrayKeys are auto-assigned numbers starting from 0
🔑 Associative ArrayKeys are custom strings you define
🧮 Multidimensional ArrayArrays that contain other arrays (2D, 3D…)

📌 Indexed Arrays

Elements are stored with numeric keys starting from 0. PHP assigns these automatically when you use array().

Visual: $colors = array("Red", "Green", "Blue")

Index [0]
Red
Index [1]
Green
Index [2]
Blue

📝 Example 1 — Create with array()

<?php
$colors = array("Red", "Green", "Blue");

// Access each element by its index number
echo "Colors are " . $colors[0] . ", " . $colors[1] . " and " . $colors[2];
?>
Output:Colors are Red, Green and Blue

📝 Example 2 — Assign elements one by one (heterogeneous)

<?php
$array[0] = "abc";   // String
$array[1] = 'a';    // Character
$array[2] = 100;    // Integer
$array[3] = 200;    // Integer

// Arrays can hold mixed types — strings AND numbers together
print "first " . $array[0] . " second " . $array[1]
    . " third " . $array[2] . " fourth " . $array[3];
?>
Output:first abc second a third 100 fourth 200

🔑 Associative Arrays

Associative arrays use custom string keys instead of numbers. This lets you create a meaningful relationship between the key and its value — like a dictionary or a lookup table.

Syntax: Use => to pair a key with its value: "key" => "value"

Visual: Age lookup table

"Peter"
=>
"35"
"Ben"
=>
"37"
"Joe"
=>
"43"

📝 Example 1 — Mixed key types

<?php
$array = array(
    "foo"  => "bar",   // String key → String value
    "bar"  => "foo",   // String key → String value
    100    => -100,   // Integer key → Integer value
    -100   => 100    // Negative key → Integer value
);

print "first "  . $array['foo']
    . " second " . $array['bar']
    . " third "  . $array[100]
    . " fourth " . $array[-100];
?>
Output:first bar second foo third -100 fourth 100

📝 Example 2 — Assign one by one

<?php
$age['Peter'] = "35";
$age['Ben']   = "37";
$age['Joe']   = "43";

echo "Peter is " . $age['Peter'] . " years old";
?>
Output:Peter is 35 years old

🧮 Multidimensional Arrays

A multidimensional array is an array that contains other arrays as its elements. To access an element you need multiple indices — one per dimension.

2D Array: Needs 2 indices — $array[row][col]
3D Array: Needs 3 indices — $array[x][y][z]

📝 Example 1 — Building a 2D array

<?php
$row0 = array(1, 2, 3);
$row1 = array(4, 5, 6);
$row2 = array(7, 8, 9);

$multi = array($row0, $row1, $row2);

echo $multi[0][0];   // → 1 (row 0, col 0)
echo $multi[1][2];   // → 6 (row 1, col 2)
echo $multi[2][1];   // → 8 (row 2, col 1)
?>
Output:1   6   8

📝 Example 2 — Nested Associative Array (Arrays of Arrays)

Array elements can themselves be arrays — useful for structured data like a product catalogue.

<?php
$products = array(
    'paper' => array(
        'copier' => "Copier & Multipurpose",
        'inkjet' => "Inkjet Printer",
        'laser'  => "Laser Printer",
        'photo'  => "Photographic Paper"
    ),
    'pens' => array(
        'ball'   => "Ball Point",
        'hilite' => "Highlighters",
        'marker' => "Markers"
    ),
    'misc' => array(
        'tape'  => "Sticky Tape",
        'glue'  => "Adhesives",
        'clips' => "Paperclips"
    )
);

// Access: category → sub-key
echo $products["pens"]["marker"];   // → Markers
echo $products["paper"]["laser"];   // → Laser Printer
?>
Output:Markers
Laser Printer

🔁 Traversing Arrays

The most common task with arrays is to do something with every element — e.g., display data, send emails, or calculate totals. The method you use depends on the array type.

1. Loop Through Indexed Array — for loop

Use count() to get the array length, then loop through each index.

<?php
$color     = array("Red", "Blue", "Green");
$arrlength = count($color);   // → 3

for ($x = 0; $x < $arrlength; $x++) {
    echo $color[$x] . "<br>";   // Access by index each time
}
?>
Output:Red
Blue
Green

2. Loop Through Associative Array — foreach loop

foreach is the most natural way — it gives you both the key and value on each iteration.

<?php
$marks = array("Ruwan" => 35, "Saman" => 37, "Ravi" => 43);

foreach ($marks as $x => $x_value) {
    echo "Student Name = " . $x . ", Marks = " . $x_value . "<br>";
}
?>
Output:Student Name = Ruwan, Marks = 35
Student Name = Saman, Marks = 37
Student Name = Ravi, Marks = 43

3. Loop Through Multidimensional Array — Nested for loop

Use an outer loop for rows and an inner loop for columns.

<?php
$shop = array(
    array("rose",   1.25, 15),
    array("daisy",  0.75, 25),
    array("orchid", 1.15,  7)
);

for ($row = 0; $row < 3; $row++) {      // Outer loop = rows
    echo "<b>Row $row:</b><ul>";
    for ($col = 0; $col < 3; $col++) {   // Inner loop = columns
        echo "<li>" . $shop[$row][$col] . "</li>";
    }
    echo "</ul>";
}
?>
Output: Row 0: rose · 1.25 · 15
Row 1: daisy · 0.75 · 25
Row 2: orchid · 1.15 · 7

🔧 Array Functions

count( $array ) — Get Length
Returns the total number of elements in an array. 
<?php
$cars = array("Volvo", "BMW", "Toyota");
echo count($cars);   // → 3
?>
Output:3
array_sum( $array ) — Sum All Values
Adds up all numeric values in an indexed or associative array and returns the total. 
<?php
$scores = array(98, 76, 56, 80);
$total  = array_sum($scores);   // 98 + 76 + 56 + 80
echo "Total: $total";
?>
Output:Total: 310
array_push( $array, $val1, $val2, ... ) — Add to End
Inserts one or more elements to the end of an array. The array grows dynamically. 
<?php
$a = array("red", "green");
array_push($a, "blue", "yellow");   // Add 2 elements at once
print_r($a);
?>
Output:Array ( [0] => red [1] => green [2] => blue [3] => yellow )
unset( $array[key] ) — Delete an Element
Removes a specific element from an array by its key. The remaining elements keep their original keys (indices do not re-number automatically). 
<?php
$anArray = array("X", "Y", "Z");

unset($anArray[0]);   // Remove element at index 0 ("X")
print_r($anArray);     // Index 0 is gone; 1 and 2 remain
?>
Output:Array ( [1] => Y [2] => Z )
⚠️ Note: After unset(), the array indices do not shift. Index [1] and [2] stay as-is. Use array_values($array) afterwards if you need to re-index from 0.
sort( $array ) — Sort Array
Sorts an array in ascending order. Modifies the original array directly. 
<?php
$nums = array(5, 2, 8, 1, 4);
sort($nums);
print_r($nums);
?>
Output:Array ( [0] => 1 [1] => 2 [2] => 4 [3] => 5 [4] => 8 )

🔍 Inspecting Arrays

PHP provides functions to check properties of arrays — whether a variable is an array, or whether a specific value exists inside one.

FunctionWhat It DoesReturns
is_array($var) Checks if a variable is an array true or false
in_array($val, $arr) Checks if a value exists in an array true or false
isset($array[$key]) Checks if a specific key exists in an array true or false

📝 Example — is_array()

<?php
$a = "Hello";
echo "a is array: " . (is_array($a) ? "Yes" : "No") . "<br>";   // No

$b = array("red", "green", "blue");
echo "b is array: " . (is_array($b) ? "Yes" : "No") . "<br>";   // Yes

$c = array("Peter" => "35", "Ben" => "37");
echo "c is array: " . (is_array($c) ? "Yes" : "No") . "<br>";   // Yes

$d = "red, green, blue";   // Just a string, not an array
echo "d is array: " . (is_array($d) ? "Yes" : "No") . "<br>";   // No
?>
Output: a is array: No
b is array: Yes
c is array: Yes
d is array: No

📝 Example — in_array()

<?php
$people = array("Peter", "Joe", "Glenn", "Cleveland");

if (in_array("Glenn", $people)) {
    echo "Match found";        // "Glenn" exists → this runs
} else {
    echo "Match not found";
}

echo "<br>";

if (in_array("Alice", $people)) {
    echo "Match found";
} else {
    echo "Match not found";    // "Alice" doesn't exist → this runs
}
?>
Output:Match found
Match not found

📝 Example — isset()

<?php
$age = array("Peter" => 35, "Ben" => 37);

if (isset($age["Peter"])) {
    echo "Peter key exists: " . $age["Peter"];   // Key found → prints 35
} else {
    echo "Peter key not found";
}
?>
Output:Peter key exists: 35
