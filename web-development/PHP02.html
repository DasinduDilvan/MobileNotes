🔁 PART 1 — PHP Loops

💡 What Are Loops?

Loops execute a block of code a specified number of times, or while a specified condition is true. They eliminate repetitive code by automating repeated tasks.

PHP supports four looping statements:

whileChecks condition before each run. May never execute if condition starts false.
do...whileChecks condition after each run. Always executes at least once.
forUsed when the exact number of iterations is known in advance.
foreachDesigned for arrays. Iterates over every element automatically.

🔄 while Loop

The while loop executes a block of code repeatedly as long as the condition remains true. The condition is evaluated before each iteration.

Key Concepts

  • Condition is checked before the body executes each time
  • If the condition is false at the start, the body never runs (zero iterations)
  • The variable in the condition must be updated inside the loop body
  • Failing to update the condition variable causes an infinite loop
  • Best used when the number of iterations is not known in advance
Syntax: while (condition) { // body }

Loop Anatomy

$num = 1; ① initialise before loop while ($num <= 5) ② condition checked here — BEFORE each run {     echo "The number is " . $num . "<br>";     $num++; ③ update — prevents infinite loop! }
⚠️ Infinite Loop: If the condition variable is never updated, the condition never becomes false and the loop runs forever — crashing the server.

🔃 do...while Loop

The do...while loop executes the body first, then checks the condition. The body is guaranteed to run at least once regardless of the condition.

Key Concepts

  • Body executes first — condition is checked after each iteration
  • Guarantees the body runs at least one time
  • Note the mandatory semicolon after the closing while parenthesis: while (condition);
  • Useful for menus, prompts, or situations where an initial execution is always required
Syntax: do { // body } while (condition);

Critical Difference from while

while — condition BEFORE
  • ▸ Condition checked before first run
  • ▸ If false from start → body never runs
  • ▸ Minimum iterations: 0
VS
do...while — condition AFTER
  • ▸ Body runs first, then condition checked
  • ▸ Even if false from start → body always runs once
  • ▸ Minimum iterations: 1

🔢 for Loop

The for loop is used when you know in advance exactly how many times the script should run. It combines initialisation, condition, and increment in a single header line.

Key Concepts

  • Three parts in the header: initialise ; condition ; increment
  • Initialise — runs once at the very start
  • Condition — checked before every iteration; loop stops when false
  • Increment/Decrement — runs after every iteration
Syntax: for (init; condition; increment) { // body }

Loop Anatomy — Annotated

for ( $start = 0 ① init — runs once ; $start < 11 ② condition — checked before each run ; $start++ ③ increment — runs after each run ) {     $counter = $counter + 1;     print $counter . "<BR>"; }

📋 foreach Loop

The foreach loop is designed specifically to iterate over every element of an array. It loops automatically without needing a counter variable. Elements are processed in their internal order.

Key Concepts

  • Specifically designed for arrays — not suitable for non-array values
  • No counter variable needed — PHP manages iteration automatically
  • Two forms: with key-value pair, or value only
  • $key holds the index (number) or string key of each element
  • $value holds the actual data of each element
  • Elements are processed in their internal (insertion) order
With key:
foreach ($arr as $key => $value) { }
Use when you need both the key and value.
Value only:
foreach ($arr as $value) { }
Use when you only need the element data.

🔳 Nested Loops & Modulo Operator

Nested Loops

A nested loop is a loop placed inside another loop. For each single iteration of the outer loop, the inner loop runs through all its iterations completely.
  • Total iterations = outer count × inner count (e.g., 10 × 10 = 100)
  • Commonly used for grids, matrices, and multi-dimensional combinations
  • Each loop has its own counter variable (e.g., $a and $b)
Two-digit combinations example:
Outer loop controls tens digit (0–9) · Inner loop controls units digit (0–9)
Result: 10 × 10 = 100 combinations (00 to 99)

Modulo Operator %

The modulo operator % returns the remainder of a division. If $x % $n == 0, then $x is perfectly divisible by $n (no remainder).
  • Used to check divisibility — essential for FizzBuzz-style problems
  • Order of checks matters in FizzBuzz: always check the combined condition (%3==0 && %5==0) first before individual ones
ExpressionMeaningFizzBuzz Use
$i % 3 == 0Divisible by 3Print "Fizz"
$i % 5 == 0Divisible by 5Print "Buzz"
$i % 3 == 0 && $i % 5 == 0Divisible by bothPrint "FizzBuzz" — check first!

🗂️ PART 2 — PHP Arrays

💡 What Is an Array?

An array is a special variable that can hold more than one value at a time. It is an ordered collection of key-value pairs where each stored value is called an element. Arrays can hold heterogeneous data — strings, integers, booleans, other arrays simultaneously.
📌 Indexed ArrayKeys are auto-assigned numbers starting from 0.
🔑 Associative ArrayKeys are custom strings defined by the programmer.
🧮 Multidimensional ArrayArrays that contain other arrays as elements.
Heterogeneous data: An array is not limited to one type. A single array can hold strings, integers, booleans, and other arrays at the same time.

📌 Indexed Arrays

Indexed arrays store elements with automatically assigned numeric keys starting from 0. Elements are accessed by their position (index number).

Key Concepts

  • Index numbers always begin at 0 and extend to length − 1
  • Created using array() or by direct assignment: $arr[0] = "value"
  • Elements accessed using square brackets: $array[index]
  • Can store mixed types — strings and numbers in the same array

🔑 Associative Arrays

Associative arrays use custom string keys (instead of auto-numbers) to create a meaningful relationship between key and value — like a dictionary or lookup table. Keys and values are paired using the => operator.

Key Concepts

  • Functionally similar to indexed arrays — differ only in key type
  • Keys are custom strings defined by the programmer
  • Creates a strong association between a key name and its value
  • Keys can also be integers (including negative integers)
  • Elements accessed using string keys: $array["keyname"]

🧮 Multidimensional Arrays

A multidimensional array is an array that contains other arrays as its elements. The dimension equals the number of indices needed to access one element.

Key Concepts

  • Values in an array can themselves be arrays
  • 2D array — needs 2 indices: $arr[row][col]
  • 3D array — needs 3 indices: $arr[x][y][z]
  • Array elements can be strings, integers, objects, or other arrays
  • Can combine indexed and associative levels: $products["pens"]["marker"]

🔁 Traversing Arrays

The most common task with arrays is to do something with every element. The traversal method depends on the array type and the task being performed.

1. for Loop — Indexed Arrays

  • Use count() to get the array length as the loop limit
  • The for loop processes elements in key order
  • Access each element using the loop counter as the index: $array[$x]
for ($x = 0; $x < count($array); $x++) { echo $array[$x]; }

2. foreach — Associative Arrays

  • The most common way to loop over associative arrays
  • Elements are processed in their internal order
  • Gives access to both key and value on each iteration
foreach ($marks as $x => $x_value) { echo $x . ": " . $x_value; }

3. Nested for Loops — Multidimensional Arrays

  • Outer loop iterates through rows
  • Inner loop iterates through columns
  • Access each element using both loop counters: $shop[$row][$col]
for ($row = 0; $row < 3; $row++) {
  for ($col = 0; $col < 3; $col++) {
    echo $shop[$row][$col];
  }
}

🔧 Array Functions

count( $array )Get length
Returns the total number of elements in an array. Used to control for loop iteration limits.
array_sum( $array )Sum values
Adds up all numeric values in an indexed or associative array and returns the total. Syntax: $total = array_sum($array);
array_push( $array, $val1, $val2, ... )Add to end
Inserts one or more elements to the end of an array. The array grows dynamically. Returns the new array length.
unset( $array[key] )Delete element
Removes a specific element by its key. The remaining elements keep their original keys — indices do NOT re-number automatically after deletion.
After unset($arr[0]), index [0] is gone but [1] and [2] remain as-is.
sort( $array )Sort ascending
Sorts an array in ascending order. Modifies the original array directly.
current() · next() · reset()Internal pointer
  • current($arr) — returns the element at the current internal pointer position
  • next($arr) — advances the internal pointer forward by one
  • reset($arr) — moves the internal pointer back to the first element

Array Inspection Functions

is_array( $var )Check if array
Takes any variable and returns true if it is an array, false otherwise. Works on strings, numbers, arrays, etc.
in_array( $value, $array )Check if value exists
Checks if a given value exists as an element in the array. Returns true if found, false if not.
isset( $array[$key] )Check if key exists
Returns true if the specified key exists in the array AND its value is not NULL. Checks for key existence, not value content.

📌 Glossary & Final Summary

Key Definitions

Loop
A control structure that executes a block of code repeatedly — either a specified number of times or while a condition is true. Eliminates repetitive code.
while Loop
Checks condition before each run. May never execute (0 minimum runs). Update condition variable inside loop or risk infinite loop. Best for unknown iteration counts.
do...while Loop
Runs body first, then checks condition. Always executes at least once (1 minimum run). Ends with a semicolon after the while clause.
for Loop
Has three parts in header: initialise (runs once); condition (checked before each run); increment (runs after each run). Best when number of iterations is known.
foreach Loop
Designed for arrays. Iterates over every element automatically. No counter needed. Two forms: with key=>value pair, or value only. Elements processed in internal order.
Infinite Loop
A loop whose condition never becomes false — runs forever and crashes the server. Caused by failing to update the loop condition variable inside the loop body.
Nested Loop
A loop placed inside another loop. Inner loop completes all its iterations for every single iteration of the outer loop. Used for grids and multi-dimensional data.
Array
A special variable holding more than one value at a time. An ordered collection of key-value pairs. Each stored value is an element. Can hold heterogeneous data types.
Element
An individual value stored inside an array. Each element has a key (index) that identifies its position or name within the array.
Index
The positional key of an element in an indexed array. Always starts at 0 for the first element and extends to length − 1 for the last.
Associative Array
An array whose keys are meaningful custom strings defined by the programmer. Uses => to pair keys with values. Creates a strong name-value association.
Multidimensional Array
An array containing one or more other arrays as elements. The number of dimensions equals the number of indices needed to access one element. 2D needs 2 indices; 3D needs 3.
Heterogeneous Array
An array holding values of different data types simultaneously — e.g., strings and integers in the same array. PHP arrays are not restricted to a single type.
unset() — No Re-indexing
After deleting an element with unset(), remaining elements keep their original keys — indices do NOT automatically shift. Use array_values() to re-index from 0 if needed.
